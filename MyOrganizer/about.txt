Pet-project: "My Organizer"

1.  Программа-органайзер, которая отображает задачи.
    Каждая задача состоит из следующих пунктов:

        * id задачи (уникальный номер)
        * название задачи
        * описание задачи
        * дата начала выполнения задачи (в формате dd.mm.yyyy)
        * дата окончания выполнения задачи (в формате dd.mm.yyyy)
        * текущий прогресс (число от 0 до 100%)

    QML файлы:

        * Основное окно (main.qml)
        * Рабочая область
        * Делегат
        * Модель (заполнить начальными данными)

2.  Добавлен пользовательский виджет - кнопка в отдельном файле. В рабочей
    области размещена кнопка "Добавить", которая добавляет новую задачу.

3.  Создано окно для добавления нового задания. 
    В этом окне есть поля для ввода составляющих, которые есть у любой задачи, 
    кроме id (он генерируется автоматически). 
    Добавлена валидация пользовательского ввода (текущий прогресс - целое число от 0 до 100).

4.  Добавлен ввод даты начала и окончания выполнения задачи через виджет календаря.
