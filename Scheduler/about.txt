
Pet-project: "Scheduler v. 1"

 Программа "Планировщик запуска приложений". 
 Планировщик оформлен в виде таблицы:
   Имя программы	Расположение
   Name         	/somedir/progname

 Для добавления новой программы нажмите Add (+).

 Для удаление старой программы нажмите Delete (Del).

 Чтобы редактировать "Имя программы" включите режим редактирования действием Edit (Ctrl+E).

 Для запуска программы дважды кликните по строке содержащей необходимую программу, убедитесь при этом, что режим редактирования выключен.

 В программу добавлен настоящий файл описания, размещенный в ресурсах программы. Для вызова описания нажмите кнопку "О программе" (F1).

 Для выхода из планировщика нажмите Exit, или закройте окно.

 Для сохранения текущих настроек планировщика нажмите Save (Ctrl+Shift+S).

 В случае если в дирректории планировщика будет обнаружен файл настроек ("settings.set") данные из него будут считаны при запуске планировщика.

 Для запуска приложений из строк 2-10 планировщика воспользуйтесь горячими клавишами F2-F10.
